# Reto3IA
La problemática a resolver es el saber que un pedido fue entregado completo y de manera correcta para una tienda de regalos, se tendría que empatar la foto que se suba a la app y la que se tiene como referencia para ver si contiene el arreglo florar, el oso de peluche y los chocolates; o bien si tiene la caja de snacks y el globo al cliente adecuado.
